# 每日回顧

互動式的一天回顧，在 IDE 中使用。比 daily-journal 更即時、更對話式。

## 觸發方式

- IDE 內：`/daily-review`

## 執行步驟

### 1. 收集今日資料
- 讀取今日的 L1 工作日誌：`~/work-logs/YYYY/MM/YYYY-MM-DD.md`
- 讀取今日的 L2 精煉日記（如果已產生）：`100_Journal/daily/YYYY-MM-DD.md`
- 查詢 Dayflow 資料庫中今日的活動記錄（如果可用）

### 2. 產出回顧摘要
向 Dex 呈現：

```
## 今日回顧 — YYYY-MM-DD

### 完成了什麼
- ...（從 L1/L2 提取）

### 花最多時間在
- ...（從 Dayflow 時間軸推斷）

### 值得深入的點
- ...（從對話和反思中萃取）

### 未完成 / 明天要繼續
- ...（從待辦清單提取）
```

### 3. 互動對話
回顧完後，問 Dex：
- 「今天有什麼你覺得特別值得記下來的想法嗎？」
- 「有哪個洞察你想發展成內容？」
- 「明天最重要的一件事是什麼？」

### 4. 更新日記
如果 Dex 補充了額外想法：
- 更新 `100_Journal/daily/YYYY-MM-DD.md` 的洞察區塊
- 如果有值得發展的主題，建議跑 `/topic-create`

## 注意事項
- 這是互動流程，不是自動化腳本
- 語氣像一個幫你整理思緒的助手，不是考官
- 不要問太多問題，3 個就夠
- 繁體中文
