# 原則 vs 規則：AI Agent 設計的核心框架

給 Agent 寫 rules 時，原則（方向性指引）比規則（硬性約束）更能應對邊界情境，這跟「建立觀點而非清單」一脈相通。

## 潛在切入角度
Threads 用一個具體的 agent rule 改寫前後對比（規則版 vs 原則版）；Newsletter 延伸到「為什麼寫內容也是原則優先」，串接 Threads 工作坊的框架
