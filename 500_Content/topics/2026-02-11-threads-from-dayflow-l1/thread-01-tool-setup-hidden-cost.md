今天花了 1.5 小時裝兩個 CLI 工具

結果一個還沒裝成功

npm 權限錯誤 → 改 brew → PATH 吃不到
Node 版本不對 → nvm 切版 → 還是要手動補 PATH

你以為裝個工具五分鐘的事
實際上是一場 debug 馬拉松

最諷刺的是
這 1.5 小時完全沒有產出任何「作品」
沒寫到一行業務邏輯
沒推進任何專案進度

但你不做又不行
因為工具是後面所有產出的前提

這就是工程師最不被看見的成本

老闆看你今天產出了什麼
你說「我在配環境」
對方臉上那個表情.....你懂的

我現在的原則是
環境配置如果超過 30 分鐘還沒搞定
就先放下 改用替代方案
不要跟工具較勁

你的時間比任何一個 CLI 都值錢

下次有人問你「這個功能很簡單吧」
記得把環境建置的時間也算進去
