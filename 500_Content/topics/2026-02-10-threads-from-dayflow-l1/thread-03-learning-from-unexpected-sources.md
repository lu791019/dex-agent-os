講一個很反直覺的事

我昨天一個重要的技術發現
來源不是技術部落格
不是官方文件
是 IG Reels

對就是那個你以為只能看貓咪跟迷因的地方

我滑到一個 Claude Code 插件推薦

然後一路深挖下去收穫超多

看起來很分心對吧

但這就是探索期該有的樣子
你不知道靈感會從哪裡冒出來

別再只從「正經管道」找資訊了
