# Style DNA 作為個人品牌工具

用 LLM 分析自己的歷史內容來萃取風格指紋，再用這個指紋生成新內容——形成「分析→萃取→生成→再分析」的正向循環，這個 loop 很有教學價值。

## 潛在切入角度
Newsletter 適合寫完整教學流程（步驟拆解 + prompt 範例），Blog 可以做深度版含技術實作細節，強調「AI 不是取代你的風格，是放大你的風格」
