# Skill 不是工具，是知識管理的改革

多數人把 Skill 當 function call，但真正的價值在於將隱性知識結構化為可觸發的協議。這句話重新定義了 AI agent 的設計思維。

## 潛在切入角度
用「function call vs 知識協議」的對比開場，舉一個具體 Skill 範例說明隱性知識如何被結構化，結尾拋出「你的工作流中有哪些隱性知識可以變成 Skill？」
