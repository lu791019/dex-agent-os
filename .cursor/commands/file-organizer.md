
# File Organizer

Personal organization assistant for maintaining clean, logical file structures without the mental overhead of manual organization.

## What This Skill Does

1. **Analyzes current structure**: Reviews folders/files to understand what you have
2. **Finds duplicates**: Identifies duplicate files across directories
3. **Suggests organization**: Proposes logical folder structures based on content
4. **Automates cleanup**: Moves, renames, and organizes with your approval
5. **Reduces clutter**: Identifies old files you probably don't need

## Quick Usage

```
Help me organize my Downloads folder
```

```
Find duplicate files in my Documents folder
```

```
Review my project directories and suggest improvements
```
