
# Meeting Insights Analyzer

Transforms meeting transcripts into actionable insights about communication patterns. Identifies recurring behaviors (conflict avoidance, filler words, interruptions, speaking ratios) and provides timestamped examples with concrete improvement suggestions.

## When to Use

- Analyzing communication patterns across multiple meetings
- Getting feedback on leadership and facilitation style
- Identifying conflict avoidance or indirect communication
- Understanding speaking habits, filler words, and listening gaps
- Tracking communication improvement over time
- Preparing performance reviews with concrete examples

## What This Skill Does

1. **Pattern Recognition** -- Finds recurring behaviors: conflict avoidance, speaking ratios, turn-taking, question-asking patterns, active listening indicators
2. **Communication Analysis** -- Evaluates clarity, directness, filler words, hedging language, tone, and facilitation effectiveness
3. **Actionable Feedback** -- Provides timestamped examples with what happened, why it matters, and how to improve
4. **Trend Tracking** -- Compares patterns over time when analyzing multiple meetings

## Quick Start

Put meeting transcripts (.txt, .md, .vtt, .srt) in a folder, then ask:

```
Analyze all meetings in this folder and tell me when I avoided conflict.
```

```
Look at my meetings from the past month and identify my communication patterns.
```

```
Compare my facilitation style between these two meeting folders.
```
