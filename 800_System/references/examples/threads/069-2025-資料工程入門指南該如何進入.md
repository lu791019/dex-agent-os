## 原文
2025 資料工程入門指南：該如何進入

很常被「學 SQL & Python」這個建議炸得懷疑人生
也很常發現自己迷失在 Spark、Flink、BigQuery、Iceberg 技術

其實只要把最關鍵的和精要的學起來就好

1️⃣ SQL & Python  不用全部懂，掌握精要的部分
2️⃣ 分散式計算技術  熟悉 Spark、BigQuery 或 Snowflake
3️⃣ 數據管道 & 排程工具  了解 Airflow、Mage 或 Databricks Workflows
4️⃣ 數據建模與數據品質管理  能處理數據 ETL，確保數據正確性
5️⃣ 作品集  需要能夠展示自己能力的專案

今天聊聊第1~3點，週五在繼續講資料治理和作品集整理

首先對於基礎能力，我們學習最精要的部分就好

SQL掌握幾個重點：

- JOIN 操作
- GROUP BY 
- 分區 vs. 索引

Python的重點：

- Pandas 處理數據
- 寫 Airflow DAG 來排程數據管道
- 調用 REST API 來擷取數據

大家覺得只要用學哪些關鍵應用就好？
