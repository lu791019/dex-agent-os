「沒有 API 就不能整合」這句話我聽太多次了。

但事實是，大部分工具之間都有中介層可以串。

我最近在接 Podwise 的資料，它本身沒開 API。
一般人到這步就放棄了。

但它能同步到 Notion 跟 Readwise。
而這兩個都有 API。

所以真正的問題從來不是「這工具有沒有 API」，
而是「你有沒有看到中間那條路」。

工程師的核心能力不是會寫程式，
是會繞路。

你繞過最離譜的一條路是什麼？
