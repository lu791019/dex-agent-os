# 回顧 — 快速每日回顧

即時回顧今天做了什麼、學了什麼，直接印在對話中，不產生檔案。

## 觸發方式

- IDE 內：`/回顧`
- 觸發詞：`回顧`（見 `30-daily-capture.md`）

## 執行步驟

### Step 1：收集資料

讀取以下來源（有什麼讀什麼，缺的跳過）：

1. `000_Inbox/daily/DATE.md`（今日素材池）
2. `~/work-logs/YYYY/MM/DATE.md`（L1 工作日誌）
3. `100_Journal/daily/DATE.md`（L2 精煉日記）
4. `100_Journal/daily/DATE-dayflow.md`（Dayflow 摘要）

### Step 2：印出結構化回顧

```
## 今日回顧 — DATE

### 做了什麼
- ...（從 L1/L2/素材池提取）

### 學了什麼
- ...（從素材池心得 + readings 提取）

### 明天要做
- ...（從待辦/未完成項目提取）

### 今天最值得記住的一件事
> ...（一句話）
```

### Step 3：互動

問 Dex：「想補充什麼嗎？有什麼想記住的？」

如果 Dex 有補充 → 印出更新版，但不存檔。

## 原則

- 不存檔 — 這是即時對話用的快速回顧
- 不產生任何檔案
- 語氣像幫你整理思緒的助手
- 快速、精簡，不要問太多問題
